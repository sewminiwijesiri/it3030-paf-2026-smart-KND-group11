# UniFlow System

## Description
Smart Campus Operations System for managing:
- Resources (rooms, labs, equipment)
- Bookings
- Incident tickets

## Tech Stack
- Spring Boot (Backend)
- MongoDB Atlas
- React (Frontend)

## How to Run Backend
1. Open project
2. Run:
   mvnw.cmd spring-boot:run

3. Access:
   http://localhost:8081

## Database
MongoDB Atlas (cloud)

## Team Modules
- Member 1: Resources
- Member 2: Booking
- Member 3: Tickets
- Member 4: Notifications
